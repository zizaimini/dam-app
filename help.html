Dam 帮助文档

Dam 是一款 macOS 菜单栏工具，专为需要在公共场合或会议中使用电脑的人设计。当你连接投影仪、接入会议共享，或按下快捷键时，Dam 自动关闭或最小化你指定的敏感应用，保护你的隐私。

🖥️
屏幕智能识别
连接已标记的"公共"屏幕时，自动触发保护。记住每块屏幕的策略，一次配置，永久生效。
🔒
应用自动保护
把需要保护的 App 加入列表，Dam 触发时一键关闭或最小化全部。
⌨️
全局快捷键
任何时候按下快捷键，立即执行保护动作，无需 Dam 在前台。
📹
会议共享检测
检测 Zoom、腾讯会议等的屏幕共享，及时提醒并触发保护。
✅
Dam 完全本地运行，无需账号，无网络依赖，所有数据只存在你的 Mac 上。

快速上手

完成以下四步，Dam 就可以正常工作了。

  1. 安装并启动 Dam
    将 Dam 拖入「应用程序」文件夹，双击启动。Dam 会出现在菜单栏右上角，图标常驻后台。
  2. 完成权限授权
    首次启动后，Dam 会打开「设置 → 权限」页面。点击各权限旁的授权按钮，按系统提示完成授权。三项权限全部变为绿色即表示完成。详见 授权说明。
  3. 添加需要保护的应用
    进入「设置 → 应用列表」，点击「+」按钮，从当前正在运行的 App 中选择，或手动输入 Bundle ID。建议把常用的邮件、聊天、银行、笔记等 App 都加进来。
  4. 接入外部屏幕 / 按下快捷键，体验保护
    连接外接显示器时，Dam 会弹出全屏提示，选择「Public（公共）」即触发保护。也可以直接按 ⌃⌥⌘W 立即关闭所有受保护 App。

授权说明

Dam 需要三项系统权限才能完整运行。设置界面会实时显示每项权限的状态。

权限 用途 如果不授权
本地网络 扫描局域网中的 AirPlay 设备，在设置界面展示周围可用的投屏设备 AirPlay 设备列表不可用，其他功能正常
自动化（Apple Events） 向受保护的 App 发送「退出」指令，实现一键关闭功能 「关闭」模式失效；「最小化」模式不受影响
辅助功能（Accessibility） 操控 App 窗口，支持自定义应用切换器的窗口移动功能 自定义 App 切换器的跨屏幕移窗功能不可用
⚠️
授权完成后，如果某项权限显示为红色，通常是因为 macOS 未刷新状态。尝试点击设置界面中的刷新按钮，或重启 Dam。

如何手动检查权限

打开 macOS「系统设置 → 隐私与安全性」，在「自动化」和「辅助功能」中找到 Dam，确认已勾选。

屏幕分类（Screen Policy）

Dam 的核心机制是对每块外部显示器贴上「安全」或「公共」的标签。

首次接入新屏幕时

连接一块 Dam 没见过的显示器时，Dam 会在所有外部屏幕上弹出全屏半透明提示浮层：

  • 点击圆圈中央（Safe）→ 标记为安全屏幕，不执行任何动作
  • 点击圆圈外侧（Public）→ 标记为公共屏幕，立即触发应用保护
💡
Dam 通过屏幕的厂商、型号、序列号三个信息唯一识别每块显示器。你家里的显示器只需分类一次，以后接入时自动执行策略，不会再弹提示。

同时接入多块新屏幕

Dam 会逐一弹出提示，先处理完一块，再处理下一块。MacBook 的内建屏幕永远不触发提示。

管理已记录的屏幕策略

进入「设置 → 屏幕策略」可以：

  • 查看所有已记录的屏幕（显示厂商、型号）
  • 将屏幕从 Safe 改为 Public，或反向修改
  • 为每块屏幕添加备注（例如「公司会议室投影」）
  • 删除记录（下次接入时重新弹出分类提示）

应用保护（App Protection）

触发公共屏幕策略后，Dam 会对「应用列表」中所有正在运行的 App 执行保护动作。

两种保护模式

模式 行为 适用场景
关闭（Quit） 彻底退出应用，进程结束 隐私要求高，不希望应用继续在后台运行
最小化（Minimize） 将应用的所有窗口最小化到 Dock 需要保留会话，想会后快速恢复工作状态

在「设置 → 行为设置」中选择默认模式。快捷键可以分别绑定到两种模式，随时切换。

添加受保护应用

  1. 打开「设置 → 应用列表」
  2. 点击「+」
  3. 从当前运行的 App 列表中选择，或手动输入 Bundle ID（例如 com.apple.mail）
💡
建议把这些 App 加入保护列表：邮件、信息、微信、Slack、网银、笔记（Notion / Obsidian）、密码管理器（1Password）等。

公共场景二次确认

在「设置 → 行为设置」中可以开启「公共场景确认」开关。开启后，Dam 在执行保护动作前会再弹一次确认浮层，防止误触（例如不小心碰了外接显示器插口）。

手动触发保护

在「设置 → 应用列表」页面有手动触发按钮，无需连接外部屏幕即可立即执行关闭/最小化动作——方便测试或临时保护。

全局快捷键（Hotkeys）

Dam 提供两个全局快捷键，在任意应用中均可触发，Dam 不需要在前台。

快捷键 默认值 功能
关闭快捷键 ⌃⌥⌘W 立即关闭（Quit）所有受保护应用
最小化快捷键 ⌃⌥⌘Y 立即最小化所有受保护应用

修改快捷键

  1. 打开「设置 → 快捷键」
  2. 点击对应的快捷键输入框
  3. 按下你想要的组合键（支持 ⌘⌥⌃⇧ 任意组合 + 字母/数字/功能键）
  4. 自动保存，立即生效
⚠️
避免与系统或其他 App 的常用快捷键冲突。如果快捷键不响应，请检查是否被其他应用占用。

会议/录屏检测（Meeting Share Detection）

除了检测物理外接屏幕，Dam 还能实时检测你的屏幕是否正在被会议软件共享或录制。

Dam 可检测的四种状态

状态 含义
无共享 屏幕未被任何人看到，Dam 不做任何动作
AirPlay Only 仅有物理投屏（AirPlay / Sidecar），无软件共享
Meeting Only 仅有会议录屏（Zoom / Teams / Google Meet / 腾讯会议 等）
Both AirPlay 和会议共享同时存在

触发方式

检测到会议/录屏共享时，Dam 弹出浮层询问：「你现在是否在公共场景？」

  • 确认 → 触发应用保护
  • 取消 → 忽略，不做任何动作
ℹ️
Dam 使用 macOS 系统级 API 检测共享状态，不接入任何会议平台的账号，也不读取会议内容。

自定义应用切换器（App Switcher）

Dam 内置了一套替代 macOS 原生 ⌥Tab 的应用切换器，专为隐私场景优化。

开启方式

在「设置 → 行为设置」中打开「自定义 App 切换器」开关，重新绑定辅助功能权限即可。

使用方法

  • 按 ⌥Tab 打开切换器（全屏透明浮层）
  • 左右滚动选择目标应用
  • 上滑确认并切换到该应用

与原生 ⌥Tab 的区别

功能 macOS 原生 Dam 自定义
排序方式 固定顺序 按最近使用时间排序
隐私应用 全部显示 自动过滤受保护应用，不出现在列表
跨屏幕移窗 不支持 上滑可将窗口从其他屏幕拉到当前屏幕并居中
💡
受保护的 App 不出现在切换器中，意味着即使你在公共场合切换应用，也不会在界面上暴露这些 App 的存在。

设置界面总览

点击菜单栏 Dam 图标 → 「设置」，打开设置窗口。侧边栏共有五个标签页：

标签页 主要内容
权限 三项系统权限的状态显示 + 一键授权入口（默认打开此页）
应用列表 添加/删除受保护应用；手动触发保护动作
屏幕策略 查看已记录的屏幕；修改 Safe/Public 分类；添加备注；删除记录
快捷键 自定义关闭快捷键和最小化快捷键
行为设置 保护模式选择、公共场景确认开关、App 切换器开关

行为设置详解

保护模式

选择触发保护时的默认动作：

  • 关闭（Quit）：彻底退出应用
  • 最小化（Minimize）：窗口缩入 Dock，保留会话

注意：这里设置的是默认模式。两个快捷键可以分别绑定到「关闭」和「最小化」，随时按需选择。

公共场景确认

关闭（默认）：屏幕被标记为 Public 后立即执行保护，无需确认。

开启：执行保护动作前额外弹出确认提示，防止误触。

自定义 App 切换器

开启后，⌥Tab 被替换为 Dam 的自定义切换器。需要辅助功能权限。

常见问题

连接了外接显示器，但 Dam 没有弹出提示？

这块屏幕之前已经被分类过了，Dam 记住了它的策略，不会重复提示。

如果想重新分类：进入「设置 → 屏幕策略」，找到该屏幕，点击删除，然后重新插拔显示器即可重新触发提示。

快捷键按了没反应？

可能的原因：

1. 辅助功能权限未授权。进入「设置 → 权限」检查状态。

2. 快捷键与其他 App 冲突。尝试修改为不常用的组合，或在其他 App 的快捷键设置中移除冲突项。

3. Dam 进程未启动。检查菜单栏是否有 Dam 图标。

「关闭」模式会不会丢失我的工作？

会。Dam 使用 Apple Events 发送退出指令，效果等同于你手动 ⌘Q 退出应用。

如果不希望丢失未保存的内容，建议：

1. 改用「最小化」模式；或

2. 开启「公共场景确认」，在确认前手动保存。

Dam 可以保护浏览器标签页吗？

App 保护功能会最小化或关闭整个浏览器窗口，无法只针对特定标签。

如需精细的标签页保护，可以配合 Dam 的配套 Chrome 扩展「Privacy Tab Guard」使用——当 Dam 触发时，扩展自动隐藏你配置的隐私域名标签页，会后可一键恢复。

Dam 会上传我的数据吗？

不会。Dam 完全本地运行，没有服务器，没有账号体系，不联网（AirPlay 设备扫描仅在局域网内进行）。你的屏幕策略、应用列表、快捷键配置全部存储在本机。

如何卸载 Dam？

1. 点击菜单栏 Dam 图标 → 退出 Dam

2. 将「应用程序」文件夹中的 Dam.app 拖入废纸篓

3. 如需清除数据，可在终端运行：
rm -rf ~/Library/Application\ Support/Dam

Dam 支持哪些 macOS 版本？

Dam 要求 macOS 14.6 Sonoma 或更新版本。

我的 Mac 没有 Apple Silicon，Dam 支持 Intel 吗？

请关注官网更新。当前版本的系统要求和支持架构以下载页面为准。
