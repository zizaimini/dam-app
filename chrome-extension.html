How to Install

1

Unzip the downloaded file

Double-click DamBrowserExtension.zip to extract it. You'll get a folder — keep it somewhere permanent (don't delete it after installing).

2

Open Chrome Extensions page

In Chrome, go to chrome://extensions or open the menu → More Tools → Extensions.

3

Enable Developer Mode

Toggle on Developer mode in the top-right corner of the Extensions page.

This allows loading extensions that haven't been published to the Chrome Web Store yet.
4

Load the extension

Click Load unpacked, then select the extracted extension folder.

5

Done — Dam is active in Chrome

The Dam extension icon will appear in your toolbar. It works alongside the Dam macOS app to protect your browser tabs during screen sharing.
